## 🔍 How it works

The software uses automated scripts to modify your JVM arguments. It sets the heap size to match your computer's RAM. High heap sizes prevent the game from stuttering. The tool also updates the game's launcher profiles to ensure these changes load with every game session. These technical adjustments occur in milliseconds. The program does not change your Minecraft account or world files. It only affects the performance settings.

No. The optimizer only changes the performance settings of your local game client. It does not provide any gameplay advantages like speed boosts or item hacks. Servers classify it as a client-side performance mod.

**My game still runs slowly. What can I do?**
